# FashionShop 👗

Este é um projeto de uma landing page para uma loja de moda, desenvolvido como parte de uma atividade prática utilizando **Bootstrap 5**. A página apresenta um catálogo de produtos e um carrossel dinâmico com ofertas da semana.

## 🚀 Tecnologias Utilizadas

- **HTML5**: Estruturação de conteúdo.
- **CSS3**: Estilizações personalizadas e ajustes de layout.
- **Bootstrap 5.3**: Framework utilizado para responsividade, componentes de interface (cards, botões, grid) e interatividade (carrossel).

## ✨ Funcionalidades

- **Design Responsivo**: O layout se adapta a diferentes tamanhos de tela (desktop, tablets e smartphones).
- **Grade de Produtos**: Exibição organizada dos itens principais da loja usando o sistema de grid do Bootstrap.
- **Carrossel de Ofertas**: Um controle deslizante que destaca as promoções da semana, permitindo navegar por múltiplos produtos em um espaço otimizado.
- **Interface Intuitiva**: Uso de componentes visuais como alertas para exibição de preços e botões de ação destacados.

## 📂 Estrutura do Projeto

O projeto consiste em um arquivo principal:
- `index.html`: Contém toda a marcação, estilos internos e referências aos scripts do Bootstrap.

*Nota: O projeto depende de ativos locais (imagens) como `logo.png`, `favicon.png` e imagens de produtos (`imagem.jpg`, etc.) para a visualização completa.*

## 🔧 Como executar

1. Faça o download ou clone este repositório.
2. Certifique-se de que todas as imagens mencionadas no código estão na mesma pasta que o arquivo `index.html`.
3. Abra o arquivo `index.html` em qualquer navegador moderno.

## 📝 Observações Técnicas

- O projeto utiliza o CDN do Bootstrap para carregar os estilos e scripts necessários.
- Foi aplicado um estilo personalizado para garantir que as imagens dos produtos mantenham uma proporção adequada dentro dos cards.
- O carrossel possui ajustes específicos de largura via Media Queries para dispositivos móveis.
